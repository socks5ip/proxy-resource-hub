# 代理协议概述

代理协议是客户端与代理服务器之间通信的规则，不同协议适用于不同的网络需求和设备环境。

常见协议包括：

- **SOCKS5**
- **HTTP / HTTPS**
- **L2TP**
- **PPTP**

不同协议在性能、适用场景、加密方式和兼容性上各有特点。

# 为什么需要了解代理协议？

在使用代理IP前，你需要回答三个核心问题：

1. **你的使用场景是什么？**
   - 浏览网页、社媒运营、跨境电商、游戏加速等
2. **你需要的网络层级是什么？**
   - 应用层代理（HTTP / SOCKS5）还是系统层VPN（L2TP / PPTP）
3. **你关注的网络特性是什么？**
   - 稳定性、匿名性、加密性、延迟

根据这些需求，选择最适合的协议和IP类型才能最大化效率。

# 代理协议对比概览

| 协议 | 层级 | 加密 | 适用场景 | 兼容性 | 延迟表现 |
|------|------|------|-----------|--------|----------|

# IP类型概述

代理IP除了协议不同，还存在以下分类：

- **动态IP / 静态IP**  
  动态IP：每次连接可能变化，适合爬虫与短期账号运营  
  静态IP：长期固定，适合广告投放和跨境电商  
- **住宅IP / 机房IP**  
  住宅IP：宽带用户IP，低风控，真实用户环境  
  机房IP：云服务商IP，高速稳定，但易被识别为代理  
- **独享IP / 共享IP**  
  独享IP：唯一用户使用，稳定性高，安全性高  
  共享IP：多人共用，价格低，可能存在封禁风险

4. **机房IP + 高速协议**  
   - 游戏加速、批量爬虫

# 常见问题FAQ

## 1. SOCKS5和HTTP有什么区别？
- SOCKS5适合应用层代理，支持TCP/UDP，延迟低  
- HTTP仅支持HTTP/HTTPS请求，适合网页浏览和爬虫

## 2. L2TP和PPTP有什么区别？
- L2TP结合IPSec加密，安全性高  
- PPTP加密较弱，但连接速度快

## 3. 动态IP和静态IP怎么选择？
- 短期任务：动态IP  
- 长期运营：静态IP，降低风控风险

## 4. 住宅IP和机房IP如何判断？
- 住宅IP来自真实宽带用户，低封禁率  
- 机房IP来自云服务商，适合高速批量请求

## 5. 独享IP和共享IP哪个好？
- 独享IP稳定性高，适合重要业务  
- 共享IP成本低，适合临时任务
