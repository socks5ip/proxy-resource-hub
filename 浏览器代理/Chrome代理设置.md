# Chrome代理设置有几种方式？

目前主要有三种方案：

## 方案一：系统代理模式

适合：

* HTTP代理
* HTTPS代理

优点：

* 配置简单
* 系统直接调用

缺点：

* 功能有限
* SOCKS5认证兼容性一般

Chrome默认使用系统代理设置。

* SOCKS5代理

* Proxifier

* 有米加速器

这是目前国内代理IP用户最常见的使用方式。

---

* 多账号运营
* 防关联业务

推荐：

* 比特浏览器
* AdsPower
* 紫鸟浏览器

## 方案四：启动参数模式

Chrome支持命令行启动SOCKS5代理。

例如：

```bash
chrome.exe --proxy-server=socks5://127.0.0.1:1080
```

该方式通常用于测试环境。
