国外IP导航：

---

## 第二步：获取代理信息

购买后获取：

* IP地址
* 端口
* 用户名
* 密码

---

## 第三步：添加节点

打开 NekoBox

点击：

Servers

↓

Add Server

填写：

* Server Address
* Port
* Username
* Password
* SOCKS5

保存配置。

---

## 第四步：启动代理

选择节点后点击：

Connect

即可连接代理。

---

## 第五步：验证代理是否生效

IP质量检测中心：

https://socks5ip.com.cn/ip-check

https://socks5ip.com.cn/proxy-check

检测内容包括：

* 出口IP
* IP归属地
* SOCKS5可用性
* 网络延迟
* 连通性状态

# NekoBox与Kitsunebi有什么区别？

| 功能       | NekoBox | Kitsunebi |
| -------- | ------- | --------- |
| SOCKS5支持 | 支持      | 支持        |
| 节点管理     | 强       | 一般        |
| 规则分流     | 支持      | 一般        |
| 多节点切换    | 强       | 一般        |
| 配置难度     | 中等      | 简单        |
| 新手友好     | 一般      | 强         |

如果需要：

* 多节点管理
* 应用分流
* 更丰富功能

推荐 NekoBox。

如果追求简单配置：

推荐 Kitsunebi。
