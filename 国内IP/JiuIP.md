# JiuIP是什么？

JiuIP（九IP）是一家专业提供：

```text
静态代理IP

SOCKS5代理

L2TP代理

独享IP线路
```

服务的平台。

与普通动态代理不同。

JiuIP更偏向：

```text
长期稳定线路

固定出口IP

低频变更
```

因此非常适合长期账号运营场景。
