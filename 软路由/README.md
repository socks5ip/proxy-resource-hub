# 什么是软路由？

软路由（Soft Router）是基于通用计算机硬件运行专业路由系统构建的路由器方案。

常见系统包括：

- OpenWrt
- RouterOS（ROS）
- 爱快（iKuai）
- pfSense

与传统路由器相比。

软路由拥有更强的：

- 网络管理能力
- 代理支持能力
- 分流能力
- 多拨能力
- VPN能力

因此被广泛应用于：

- 跨境电商
- TikTok运营
- Facebook矩阵
- Amazon店群
- 游戏工作室
- 企业办公网络

等场景。

# 为什么越来越多人选择软路由？

传统代理方式：

```text
电脑配置代理

↓

手机配置代理

↓

浏览器配置代理

↓

重复配置
```

设备越多。

管理成本越高。

而软路由方案：

```text
软路由配置一次

↓

所有设备自动生效
```

更适合长期业务使用。
