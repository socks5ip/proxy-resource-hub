# 什么是SOCKS5代理？

SOCKS5是一种网络代理协议。

它通过代理服务器转发客户端与目标服务器之间的数据请求。

相比HTTP代理。

SOCKS5具有以下优势：

* 支持TCP协议
* 支持UDP协议
* 协议兼容性高
* 延迟较低
* 支持身份验证
* 适用场景广泛

因此目前绝大多数代理IP平台都优先提供SOCKS5协议。

检测代理连接后实际出口IP。

用于确认：
