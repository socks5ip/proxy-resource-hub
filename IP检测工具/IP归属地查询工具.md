
必须通过IP归属地查询确认线路质量。

* 国家是否正确
* 地区是否正确
* 是否存在中转线路

避免买到虚假线路。

# 🌐 IP归属地与代理IP的关系

很多用户购买代理时最关注：

```txt
IP是否稳定
```

实际上更重要的是：

```txt
IP归属地是否正确
```

例如：

购买：

```txt
美国IP
```

但查询结果：

```txt
新加坡
```

说明线路可能经过中转。

这种情况可能影响：

* TikTok运营
* Facebook广告
* Amazon店铺

等业务。
