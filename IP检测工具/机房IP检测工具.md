# 什么是机房IP？

机房IP（Datacenter IP）是由数据中心、云服务器或IDC机房分配的公网IP地址。

通常来自：

* 阿里云
* 腾讯云
* 华为云
* AWS
* Google Cloud
* Azure
* Vultr
* DigitalOcean

等云服务商。

检测结果中通常显示为：

```text
Datacenter
Hosting
Cloud
IDC
```

即：

```text
机房IP
数据中心IP
云服务器IP
```

---

# 什么是住宅IP？

住宅IP来源于：

```text
家庭宽带
运营商宽带
家庭网络
```

例如：

* 中国电信
* 中国联通
* 中国移动
* Comcast
* Verizon
* AT&T

等运营商分配给普通家庭用户的公网IP。

检测结果通常显示：

```text
Residential
ISP
Home Network
```

# 如何判断是不是机房IP？

通常可以通过以下信息判断。

---

## ASN信息

ASN（自治系统编号）可以显示IP归属网络。

例如：

```text
Amazon AWS
Google Cloud
Tencent Cloud
Alibaba Cloud
```

通常属于：

```text
机房IP
```

---

## 网络类型识别

检测工具一般会显示：

```text
Residential
Datacenter
Hosting
Cloud
Mobile
Business
```

如果结果显示：

```text
Datacenter
```

则通常属于机房IP。

---

## 云服务商识别

例如检测结果出现：

```text
AWS
Azure
Google Cloud
Alibaba Cloud
Tencent Cloud
```

一般都属于机房网络。

# 机房IP与住宅IP区别

| 对比项目  | 机房IP       | 住宅IP        |
| ----- | ---------- | ----------- |
| 来源    | 数据中心       | 家庭宽带        |
| 网络类型  | Datacenter | Residential |
| ASN归属 | 云服务商       | ISP运营商      |
| IP稳定性 | 高          | 较高          |
| 带宽资源  | 高          | 一般          |
| 部署成本  | 低          | 较高          |
| 常见用途  | 采集、监控      | 社媒、电商       |
