# 为什么需要IP检测？

很多用户购买代理IP后。

通常只关注：

```text
能不能连接
```

实际上更重要的是：

```text
IP类型
IP归属地
网络延迟
DNS是否泄漏
代理是否生效
出口IP是否正确
```

社区用户在选择代理时，通常会先检查 IP 类型、代理识别情况以及出口环境，再决定是否正式投入业务使用。

# 推荐检测流程

代理IP到手后建议按以下流程检测：

```text
第一步
查询出口IP

↓

第二步
检查归属地

↓

第三步
检测住宅IP/机房IP

↓

第四步
检测VPN识别情况

↓

第五步
检测DNS泄漏

↓

第六步
测试代理线路延迟

↓

正式投入业务
```

很多代理检测平台也会优先验证代理状态、协议类型、地理位置和响应时间。
