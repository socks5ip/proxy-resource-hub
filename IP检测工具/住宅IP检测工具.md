# 什么是住宅IP？

住宅IP（Residential IP）是由家庭宽带运营商分配给普通家庭用户使用的公网IP地址。

例如：

- 中国电信宽带
- 中国联通宽带
- 中国移动宽带
- 海外ISP运营商宽带

这些宽带出口产生的IP通常会被识别为：

```text
Residential IP
```

即：

```text
住宅IP
```

---

# 什么是机房IP？

机房IP通常来自：

- 云服务器
- IDC机房
- VPS服务器
- 数据中心

例如：

```text
Alibaba Cloud
Tencent Cloud
AWS
Google Cloud
Azure
```

等云服务商提供的服务器IP。

通常会被识别为：

```text
Hosting
Datacenter
Cloud
```

# 如何判断是不是住宅IP？

通常需要查看：

## 运营商信息

例如：

```text
China Telecom
China Unicom
China Mobile
Comcast
Verizon
AT&T
SoftBank
NTT
```

如果属于家庭宽带运营商。

更有可能属于住宅IP。

---

## ASN信息

ASN（自治系统编号）能够帮助判断IP来源。

例如：

```text
AS4134
China Telecom
```

通常属于运营商网络。
